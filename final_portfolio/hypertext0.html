Interactive web-based storytelling project.

Open Hypertext Experience

This project explores nonlinear storytelling through linked webpages and interactive navigation. The work was created using HTML and focuses on user exploration and internet-based narrative structures. I made this piece after deciding it would be pretty fun to make a game out of clicking on different hypertexts. So if you're a fan of Pokemon, then this is the hypertext experience for you!
